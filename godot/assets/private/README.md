# Private asset drop (gitignored)

Put downloads that are **free to use in a game build** but **not free to
redistribute as source files** here:

- RenderPeople free scans
- TurboSquid / CGTrader free RF models
- DAZ / Character Creator trial exports (until license verified)
- Meshy / Tripo / Luma generations under restrictive free tiers

Then run them through Blender → GLB and, **only if the license allows
public redistribution**, copy the cleaned GLB into
`godot/assets/models/<slot>.glb`.

See [`docs/ASSET_PIPELINE.md`](../../../docs/ASSET_PIPELINE.md) and
[`docs/ASSET_SHOPPING_LIST.md`](../../../docs/ASSET_SHOPPING_LIST.md).
